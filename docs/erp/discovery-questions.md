# ERP — вопросы для сбора требований (discovery)

Внутреннее ERP: сотрудники → цеха → сквозная прозрачность производства.
Поток: **Заказ ткани → Раскрой → Брендирование (ДТФ / Шелкография / Вышивка) → Пошив → ВТО → Упаковка**

Зафиксированные решения:
- Order Studio (ТЗ/цены) — **заархивировать за feature-flag**, вернём позже
- Новый ERP — **с чистого листа**
- Supabase — **существующий `pinhead-os-v2`**
- Ветка/деплой — **main**

Заполняй ответы прямо здесь (после «→») или в Google-таблице.

---

## A. Общая модель потока
1. Поток всегда линейный, или есть заказы, пропускающие этапы (напр. без брендирования)? →
2. Может ли партия дробиться и идти параллельно (часть на ДТФ, часть на вышивку)? →
3. Единица движения — заказ целиком / партия / изделие / пачка кроя? →
4. Есть ли возвраты между этапами (брак с пошива обратно на раскрой и т.п.)? →
5. Нужен ли учёт незавершёнки (НЗП) между этапами? →

## B. Персонал и цеха
6. Перечисли все цеха/участки и число людей в каждом. →
7. Должности внутри каждого цеха (швея, раскройщик, печатник, вышивальщица, технолог, бригадир, ОТК…)? →
8. Бригадиры/начальники цехов — что видят и что утверждают? →
9. Сотрудник закреплён за одним цехом или может работать в нескольких? →
10. Сменность (2 смены, графики)? Как учитываем смены? →
11. Идентификация в цехе — логин / QR-бейдж / табельный номер? →
12. Нужен табель присутствия (приход-уход) или только выработка? →
13. Кто заводит и редактирует сотрудников (HR / директор / начальник цеха)? →
14. Подрядчики/аутсорс-операции (напр. вышивка на стороне)? Как учитывать? →

## C. Оплата труда
15. Оплата сдельная / окладная / смешанная — по каждому цеху? →
16. Расценка за операцию / изделие / метр / штуку печати? Примеры. →
17. Кто утверждает расценки? Нужна история изменений? →
18. Нормы времени/выработки на операции? →
19. Как считается ЗП за период? Расчётный период (неделя/месяц)? →
20. Премии, штрафы, доплаты, коэффициенты (срочность, сложность)? →
21. Кто закрывает начисления и куда выгружаем (1С / банк / Excel)? →
22. Учёт брака с удержанием из ЗП? →

## D. Заказ ткани / склад сырья
23. Кто и на основании чего заказывает ткань (под заказ / на склад / min-остаток)? →
24. Параметры ткани (артикул, цвет, состав, плотность, ширина, поставщик, цена, партия/рулон)? →
25. Ведём остатки ткани и списание в раскрой? →
26. Учёт поставщиков и приёмки (приход, накладная, брак поставки)? →
27. Единицы ткани — метры / кг / рулоны? Пересчёт? →
28. Резервируется ли ткань под заказ? →
29. Учёт фурнитуры/расходников (нитки, молнии, этикетки, плёнка ДТФ, краска)? →

## E. Раскрой
30. Что фиксируем (кто, настилы, слои, расход ткани, детали, размеры)? →
31. Лекала/раскладки и норма расхода на изделие? →
32. Учёт обрезков/остатков после раскроя? →
33. Крой пачками по размерам/цветам? Маркировка пачек (QR)? →
34. Брак кроя и пересортица? →

## F. Брендирование (ДТФ / Шелкография / Вышивка)

36. От чего зависит расценка (нанесения / площадь / цвета / стежки)? →
37. Учёт расхода материалов на нанесение (плёнка, порошок ДТФ, краска, нитки)? →
38. Подготовка (макет / плёнка / сетка / программа вышивки) — отдельная операция? →
39. Несколько нанесений разными способами на одном изделии? →
40. Брак брендирования и перепечать? →
41. Внешние подрядчики по способам? →
