# PINHEAD ORDER STUDIO v1.7 — Сессия 03.03.2026

## Что сделано сегодня

### 1. PDF → HTML Print Preview
- **Убран jsPDF** — генерировал мусор вместо кириллицы
- **HTML print preview** — открывается в новом окне с кнопками «Печать / PDF» и «Закрыть»
- Кириллица отображается корректно через системные шрифты браузера
- Сохранение в PDF через Ctrl+P → «Сохранить как PDF»
- Будем дорабатывать: разбивка на разделы, подтягивание всех данных

### 2. Обработки — карточки с иконками
- Вместо скучных чекбоксов — **карточки** с SVG-иконкой, названием, описанием и ценой
- 12 уникальных иконок для каждой обработки (люверсы, шнурки, молнии, петли, швы и т.д.)
- Описания поясняют назначение каждой обработки
- Галочка в правом верхнем углу, инверсия при выборе

### 3. Express Calc → SKU каталог
- **Изделие** — выпадающий список из SKU_CATALOG, сгруппированный по категориям
- **Ткань** — живой `<select>`, автофильтрация по категории SKU
- **Крой** — убран (избыточен, уже в названии SKU)
- **Цена** — считается напрямую: пошив + ткань×расход + фурнитура×расход (с курсом $)
- «Создать заказ» → сразу выбирает нужный SKU в основном конфигураторе

### 4. Screen → Шелкография
Во всех визуальных местах:
- Кнопки выбора техники в зонах печати
- Express Calc
- Редактор цен (badge + toast-уведомления)
- Tooltip цветного текстиля

Внутренние переменные (`screenMatrix`, `getScreenSurcharge`) не тронуты.

### 5. Панель заказов → Канбан (fullscreen)
- **Полноэкранная страница** — как SKU-каталог, с тем же pe-topbar хедером
- **Канбан-доска** — 5 колонок по статусам:
  - Черновик (серый)
  - Согласование (жёлтый)  
  - Одобрен (синий)
  - В работе (оранжевый)
  - Готов (зелёный)
- **Карточки** содержат: ID, клиент, SKU + ткань, техника, тираж, сумма, дата
- **Drag & Drop** — перетаскивание между колонками меняет статус
- **Поиск** — в хедере, по всем полям (ID, клиент, SKU, артикул)
- **Статистика** — количество по статусам + общий оборот
