# PINHEAD Order Studio — Changelog

---

## [07.04.2026] — Сессия 3

### Новый функционал
- Комментарии к заказу — менеджеры могут обсуждать заказ прямо в карточке (таблица order_comments в Supabase)
- Шаблоны изделий — сохранить конфигурацию и применить к новому заказу одним кликом (таблица order_templates)
- Пагинация заказов — загрузка по 50 штук вместо 200 за раз, кнопка «Загрузить ещё»
- Папка с макетами — текстовое поле с путём к файлам на сервере + кнопка копирования
- SVG мокап изделия в печатном ТЗ — производство видит зоны нанесения визуально
- Карточка заказа расширена — мульти-позиции, контакты, менеджер, путь к макетам

### UX улучшения
- Блокировка навигации — если заказ не сохранён, система спрашивает «Остаться или уйти»
- Бирки свёрнуты в аккордеон на шаге «Дизайн» — меньше визуального шума
- Редактирование позиции ведёт сразу на шаг «Дизайн», а не «Изделие»
- Дедлайн: нельзя выбрать прошедшую дату, предупреждение если до дедлайна < 3 дней
- Индикатор заполненности шагов — галочка на заполненных шагах в прогресс-баре
- Мобильная оптимизация: свотчи компактнее, таблица размеров читаемее, фильтры на всю ширину
- Топ менеджеров в аналитике теперь группирует по менеджеру (не по клиенту)

### Документация и инфраструктура
- PINHEAD-SKILL.md — полный контекст проекта для Claude
- ACTION-PLAN.md — актуальный план задач с приоритетами
- 7 кастомных Claude Code скиллов: дебаг, TDD, архитектура, changelog, worktrees и др.

## [06.04.2026] — Сессия 1–2

### Исправления багов
- Заказ больше не теряется при ошибке Supabase — saveOrder возвращает null, черновик сохраняется
- Обновление заказа: откат данных при ошибке Supabase + null вместо фейкового объекта
- Удаление заказа: подтверждение перед удалением + ожидание ответа сервера

### Новый функционал
- Кнопка «Повторить заказ» — дублирует и сразу открывает копию в визарде
- Топ артикулов в аналитике — таблица топ-10 по количеству, заказам и сумме
- Toast-уведомление при загрузке каталогов из офлайн-кэша

### UX улучшения
- Визард сокращён с 6 до 5 шагов — обработки встроены в шаг «Изделие» как аккордеон

### Документация
- PINHEAD-PROJECT-MASTER.md — единый мастер-документ проекта (старые файлы удалены)
